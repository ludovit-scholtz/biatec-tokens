- **MICA**: Markets in Crypto-Assets (EU regulation effective 2024)
- **RWA**: Real-World Assets (physical/financial assets tokenized on blockchain)

- **AML**: Anti-Money Laundering
- **KYB**: Know Your Business (entity verification)
- **RBAC**: Role-Based Access Control
- **SSO**: Single Sign-On
- **TPS**: Transactions Per Second
- **TVL**: Total Value Locked
- **DEX**: Decentralized Exchange
- **ERC3643**: Ethereum standard for security tokens (T-REX protocol)
- **ARC**: Algorand Request for Comments (token standards)
